infrastructure as code (IAC)

pull configuration 

chef: a server,  nodes, and workstation

Chef Server manages the nodes that make up the infrastruture
Chef server has configuration in Cookbook 

Chef Client runs on each node

Knife is a command line tool on work station

workstation (local chef repo) send config to Chef Server. 
   Chef Server send config to nodes, also serves as a mirror for the nodes to compare with.
       nodes configures themself

Chef server can only run on 64-bit linux, no on 32-bit, no on windows

Chef is writen in Ruby and using Ruby as configuration language

vi hello.rb
   file 'motd' do
      content 'hello world'
   end

chef-apply hello.rb

$ cat motd
hello world

to delete motd file

vi hello.rb
   file 'motd' do
      action :delete
   end

chef-apply hello.rb

vi hello.rb
   package 'apache2' do

   service 'apace2' do
