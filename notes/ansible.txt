infrastructure as code (IAC)

developers adopts Agile Methodology, making software rollout very fast but System Admin cannot keep up with the system change.
Then it comes DevOps and Configuration Management.

Push based: Ansible, SaltStack
Pull based: Puppet, Chef

agent-based: install agent software/daemon on mananged node.
agentless:   use ssh key

LAMP stack: Linux, Apache for web server, MySql, PHP for web dev

inventory
modules, APIs, Plugins
playbook

vi lampstack.yml
---
- hosts: webservers
  tasks:
     -name: install apache2
      become_user: root   # assume root
      package: name=apache2  state=present
     -name: install apache2-php5
      become_user: root   # assume root
      package: name=apache2-php5 state=present

  handlers:
    - name: restart apache2
    service: name=apache2, state=restarted

ansible-playbook lamstack.yml

Orchestration: coordination, ordering
Provisioning: make sure that all needed softwares are installed.
