install docker

sudo apt update
sudo snap install docker

use "-s 100" in docker search  to find the most popular ones (more than 100 stars)
tian@linux1:/home/tian/tmp$ sudo docker search -s 100 centos
Flag --stars has been deprecated, use --filter=stars=3 instead
NAME                      DESCRIPTION                                     STARS               OFFICIAL            AUTOMATED
centos                    The official build of CentOS.                   5467                [OK]                
ansible/centos7-ansible   Ansible on Centos7                              122                                     [OK]
jdeathe/centos-ssh        CentOS-6 6.10 x86_64 / CentOS-7 7.6.1810 x86…   110                                     [OK]

when pulling image, make sure to specify version, eg, 'latest'. otherwise, docker will pull all images, taking away too
much space
tian@linux1:/home/tian/tmp$ sudo docker pull centos:latest
...

tian@linux1:/home/tian/tmp$ sudo docker images
REPOSITORY          TAG                 IMAGE ID            CREATED             SIZE
centos              latest              9f38484d220f        4 months ago        202MB

tian@linux1:/home/tian/tmp$ cat /etc/issue
Ubuntu 18.04.2 LTS \n \l
